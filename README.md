# My Life · 吴俊的个人网站

## 项目结构
```
my-life-website/
  index.html
  pages/ (interests/family/work/finance)
  assets/ (css/js/images)
```

## 本地预览
双击 index.html 用浏览器打开。

## 部署到 GitHub Pages
1. GitHub 创建仓库 <用户名>.github.io
2. 终端执行：
  cd my-life-website
  git init && git add . && git commit -m init
  git remote add origin https://github.com/<用户名>/<用户名>.github.io.git
  git push -u origin main
3. 访问 https://<用户名>.github.io
