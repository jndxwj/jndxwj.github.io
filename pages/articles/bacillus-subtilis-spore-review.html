枯草芽孢杆菌芽孢形成条件与控制策略综述

酱料应用 微生物 淀粉酶 📅 2026年7月

一、引言

枯草芽孢杆菌（Bacillus subtilis）是食品工业中最为常见的腐败菌之一，因其具有强大的产酶能力（尤其是淀粉酶和蛋白酶）和形成极端耐热芽孢的特性，给食品加工和储存带来持续的挑战。该菌广泛存在于土壤、灰尘、水以及植物表面，可通过多种途径进入食品生产环境。

在酱料、沙拉酱、布丁、调味品及乳制品等淀粉增稠体系中，枯草芽孢杆菌的芽孢对常规巴氏杀菌（70-85°C）具有显著耐受性。一旦芽孢在货架期萌发，营养细胞快速增殖并分泌胞外淀粉酶，会导致产品粘度急剧下降、质构劣化和分层出水等一系列质量问题。

本文基于国内外最新研究成果，对枯草芽孢杆菌芽孢形成的必要条件、分子调控机制、萌发触发因素、控制策略以及芽孢形成与淀粉酶表达之间的关系进行全面分析。

二、芽孢形成的必要条件与触发因素

2.1 营养缺乏——核心触发信号

营养缺乏是触发芽孢形成的最主要、最根本的信号。当环境中容易利用的碳源、氮源或磷源耗尽时，细菌进入稳定期，启动芽孢形成程序。

葡萄糖浓度效应：低浓度葡萄糖（约0.2%）有利于芽孢形成——枯草芽孢杆菌KM16在0.2%葡萄糖下产芽孢率可达48%。高浓度葡萄糖（>1%）通过碳代谢阻遏（CCR）抑制Spo0A的激活，从而抑制芽孢形成。

高效碳源：D-木糖、D-核糖、L-阿拉伯糖、熊果苷——12小时产芽孢率可达62-67%。高效氮源：L-脯氨酸、L-谷氨酰胺、L-天冬酰胺——产芽孢率28-82%。麸皮（3%）+ 大豆蛋白胨（3%）的配方可使产芽孢率达到96%。

关键金属离子：Mn²⁺是最重要的芽孢形成促进离子；Ca²⁺参与Ca²⁺-DPA复合物形成，是芽孢耐热性的核心保障；Mg²⁺可稳定细胞膜和核糖体。

2.2 温度的影响

芽孢形成的最适温度与营养体生长最适温度基本相同，但范围更窄：枯草芽孢杆菌XK-1的最佳产芽孢温度为40°C，产芽孢率可达96%。形成时的温度越高，成熟芽孢的热抗性越强。超过45°C时芽孢形成率显著下降。

2.3 pH的影响

最佳初始pH为7.0-7.5（中性至微碱性），pH低于5.0或高于8.5时产芽孢率显著降低。枯草芽孢杆菌BAB-1在pH 4.5-10.0的广泛范围内均能生长，具有极强的pH适应能力。

2.4 渗透压与细胞密度

高渗透压可促进芽孢形成。200 mmol/L Na⁺可使对数生长后期菌体提前到达稳定期并促进芽孢形成。枯草芽孢杆菌BAB-1可耐受1-10% NaCl、3-6% KCl、2-5% Na₂SO₄。高细胞密度有利于群体同步进入芽孢形成阶段，完整的Rap-Phr群体感应系统可在生物膜生长18小时后产生约30%的芽孢。

三、芽孢形成的分子调控

芽孢形成的核心是Spo0A磷酸传递链（phosphorelay）：多种激酶（KinA-E）感知环境信号 → Spo0F（中间载体）→ Spo0B（磷酸转移酶）→ Spo0A（主控转录因子）。Spo0A~P水平决定细胞命运：

  • 低水平Spo0A~P → 生物膜形成和同类相食（可逆）
  • 高水平Spo0A~P → 启动芽孢形成程序（不可逆）

四个Sigma因子（σF → σE → σG → σK）按精确时序在前孢子和母细胞中被依次激活，调控芽孢不同阶段的基因表达。芽孢形成全过程约需7-8小时（30°C条件下）。

四、芽孢结构与抗性特性

结构层核心组成主要功能
芽孢核心Ca²⁺-DPA（占干重5-15%）、SASP蛋白DNA保护、热稳定性、维持休眠

五、芽孢萌发的条件与机制

5.1 营养萌发受体系统

枯草芽孢杆菌芽孢内膜上有三种营养萌发受体：

受体萌发剂特点
GerAL-丙氨酸、L-缬氨酸可单独工作；D-丙氨酸可竞争抑制
GerB需GerK协同 + AGFK不能单独工作，必须与GerK协同
GerK结合葡萄糖作为共萌发剂与GerB和GerA协同

AGFK萌发体系：L-天冬酰胺（或L-丙氨酸）+ D-葡萄糖 + D-果糖 + K⁺，缺一不可。三种受体共定位于内膜的"萌发体"（germinosome）区域，由GerD脂蛋白介导组装。

5.2 萌发的两阶段过程

阶段一（萌发启动，约5-15分钟）：营养萌发剂与GRs结合 → SpoVA通道打开 → Ca²⁺-DPA从核心释放 → 核心pH从6.5降至约4.0 → 部分再水化。

阶段二（萌发后过程）：Ca²⁺-DPA释放激活皮质层裂解酶（CwlJ、SleB）→ 皮质层水解 → 核心彻底再水化 → SASP蛋白降解 → 代谢恢复 → 营养体生长。

六、芽孢形成与淀粉酶表达的关系

枯草芽孢杆菌的芽孢形成和淀粉酶表达之间既相关又相互独立，理解这一关系对设计控制策略至关重要：

  • 两者都受营养可利用性的调控，均主要在稳定期被诱导
  • 但调控路径在下游分离——decoyinine（GMP合成酶抑制剂）可在高糖条件下诱导芽孢形成，但不能诱导α-淀粉酶表达
  • spo0A（芽孢形成主控因子）直接结合mdxD（麦芽糖淀粉酶）启动子，建立代谢与分化之间的联系
  • spo0A突变导致葡萄糖对淀粉酶的抑制更严重，但该效应可通过abrB突变缓解
  • 芽孢形成的启动不是淀粉酶时间激活或去阻遏的先决条件
关键启示：抑制条件的改变对淀粉酶表达和芽孢形成的影响可能不同，控制策略可以分别针对两者设计。

七、食品体系中芽孢的控制策略

7.1 热处理控制

枯草芽孢杆菌芽孢的D值（90%杀灭时间）因食品基质不同而有显著差异：

食品基质温度D值（分钟）关键因素
菠萝汁（pH 2）95°C5.85低pH降低抗性
菠萝汁（pH 6）95°C13.70pH升高D值增大
植物基乳替代品97°C3.9-6.3基质保护效应
大豆油98.4°C53低水分—约缓冲液7倍
实验室菌株168105°C1.4食品分离株更耐热

在95°C、pH 6条件下，实现5-log降低需要约69分钟；而在pH 2下仅需约29分钟。这意味着适当调节pH可显著降低热处理要求。

7.2 高压热灭菌（HPTS/PATS）

压力辅助热灭菌是目前最有前景的芽孢灭活新兴技术，结合600-900 MPa高压和60-90°C热处理。灭活分两阶段：

  • 第一阶段：高压打开SpoVA通道，诱导Ca²⁺-DPA释放——限速步骤
  • 第二阶段：DPA释放后的芽孢变得热敏感，在中等温度下即可被灭活
  • 协同增效：HPTS + 乙醇（≤15%）或 ε-聚赖氨酸可进一步增加DPA释放和蛋白变性

7.3 防腐剂与天然抑菌剂

防腐剂有效浓度作用机制适用pH
山梨酸钾0.05-0.3%抑制细胞膜功能<6.0
苯甲酸钠0.05-0.1%破坏细胞膜透性<4.5
ε-聚赖氨酸0.01-0.05%破坏细胞膜、与DNA结合宽pH
乳酸链球菌素（Nisin）0.025-0.05%破坏细胞膜孔道<6.0

天然提取物（EPS多肽、EGCG多酚、百里香酚、大蒜素、肉桂醛、壳聚糖等）对枯草芽孢杆菌及其芽孢具有不同程度的抑制作用，其中EPS+EGCG联用展现出强α-淀粉酶抑制活性。

7.4 栅栏技术的综合应用

推荐的六道栅栏：
① 原料检测（芽孢+淀粉酶活性）+ ② 热处理（≥95°C/20-30分钟）+ ③ 酸度调节（pH<4.5）+ ④ 防腐剂（山梨酸钾0.1%）+ ⑤ 冷链（<10°C）+ ⑥ 环境监控

协同效果：以上组合可将货架期芽孢萌发率降低至单独热处理的1/100以下。

7.5 抑制产淀粉酶策略

  • 代谢调控：控制葡萄糖浓度利用CCR效应，避免使用麦芽糖（α-淀粉酶强诱导剂）
  • 天然酶抑制剂：EPS（IC50=0.162 mg/mL，强于阿卡波糖0.763 mg/mL）、EPS+EGCG协同、EDTA（螯合Ca²⁺抑制α-淀粉酶）
  • 工艺优化：分步热处理（85°C/10min + 95°C/5min）、快速通过45-20°C危险温度区间、无菌灌装
  • 萌发-杀灭循环（Germinate to Eradicate）：先通过温和热处理（45-50°C + L-丙氨酸）诱导芽孢萌发，然后在营养体状态下杀灭

八、结论与行业建议

8.1 核心结论

  • 芽孢形成由营养缺乏触发、Spo0A磷酸传递链调控——营养、温度（最适40°C）、pH（7.0-7.5）、渗透压、供氧是主要环境因素
  • 芽孢萌发由L-丙氨酸（GerA）和AGFK（GerB/GerK）触发，控制萌发剂可用性是防止货架期萌发的关键
  • 芽孢形成和淀粉酶表达在营养匮乏时被同时诱导，但下游调控路径有显著差异
  • 栅栏技术是控制芽孢最有效策略——热处理 + pH调控 + 防腐剂 + 冷链的组合可降至最低风险
  • HPTS/PATS是最有前景的新兴技术，适用于对热敏感的高品质食品

8.2 对酱料和调味品行业的实用建议

  • 建立原料淀粉酶活性数据库，对高风险原料（蜂蜜、香辛料、糖蜜、水果泥）进行常规检测
  • 优化热处理工艺：产品中心温度≥95°C维持至少20分钟（低pH产品可适当缩短）
  • 条件允许时将pH控制在4.5以下；使用交联型改性淀粉（E1442/E1412）
  • 在关键环境区域（储罐顶部、管路死角、灌装头）定期（每月至少一次）检测芽孢水平
  • 验证CIP程序对芽孢的灭活效果，将芽孢作为清洁验证的指示菌
  • 考虑在灌装前通过温和热处理（45-50°C + L-丙氨酸）诱导芽孢萌发后杀灭
